JavaScriptの基本構文と特徴

JavaScriptとは？

JavaScriptは、Webページに動的な機能を追加するために使用されるプログラミング言語です。主にクライアントサイドで使用されますが、Node.jsを使うことでサーバーサイドでも利用できます。JavaScriptは、イベントの処理、ユーザーインターフェイスの操作、Web APIとの通信など、さまざまな用途に使われます。

基本的な構文

JavaScriptにはいくつかの基本的な構文があります。これらは他のプログラミング言語にも共通するものが多く、習得することでさまざまなプログラムを記述することができます。

1. 変数の宣言

変数はデータを格納するための入れ物です。JavaScriptでは、var、let、constを使って変数を宣言します。

// varを使った変数宣言
var message = "こんにちは、JavaScript!";

// letを使った変数宣言
let count = 10;

// constを使った定数の宣言
const pi = 3.14159;

2. データ型

JavaScriptのデータ型には、string（文字列）、number（数値）、boolean（真偽値）、object（オブジェクト）、undefined（未定義）、null（無）などがあります。

3. 条件分岐

条件分岐は、特定の条件が満たされた場合に異なるコードを実行するために使われます。if文とelse文がよく使われます。

let age = 20;

if (age >= 18) {
    console.log("成人です。");
} else {
    console.log("未成年です。");
}

4. 繰り返し処理

繰り返し処理は、同じコードを複数回実行するために使用されます。forループやwhileループが一般的です。

for (let i = 0; i < 5; i++) {
    console.log("ループ番号: " + i);
}

5. 関数

関数は、特定のタスクを実行するための再利用可能なブロックです。functionキーワードを使って関数を定義します。

function greet(name) {
    return "こんにちは、" + name + "!";
}

console.log(greet("太郎"));

6. オブジェクトと配列

JavaScriptでは、オブジェクトと配列を使って複数のデータをまとめて管理することができます。

// オブジェクト
let person = {
    name: "花子",
    age: 25,
    greet: function() {
        console.log("こんにちは、" + this.name + "です。");
    }
};

// 配列
let fruits = ["りんご", "バナナ", "オレンジ"];

console.log(person.name);
person.greet();

console.log(fruits[0]);

7. テンプレートリテラル

テンプレートリテラルを使用すると、文字列内に変数や式を簡単に埋め込むことができます。バッククォート (`)を使います。

let name = "太郎";
let greeting = `こんにちは、${name}さん！`;
console.log(greeting);

8. 短絡評価

短絡評価は、論理演算子を使って簡単な条件をチェックし、最初の真または偽の値を返します。

let user = null;
let defaultUser = "ゲスト";
let name = user || defaultUser;
console.log(name); // "ゲスト" と表示される

9. 配列メソッド

JavaScriptには、配列を操作するためのさまざまなメソッドがあります。例えば、map()、filter()、reduce()などがあります。

  • map(): 各要素に対して関数を適用して新しい配列を返します。
    • let numbers = [1, 2, 3, 4, 5];
let doubled = numbers.map(function(num) {
    return num * 2;
});
console.log(doubled); // [2, 4, 6, 8, 10]
  • filter(): 条件に一致する要素だけを含む新しい配列を返します。
    • let numbers = [1, 2, 3, 4, 5];
let evenNumbers = numbers.filter(function(num) {
    return num % 2 === 0;
});
console.log(evenNumbers); // [2, 4]
  • reduce(): 配列の全要素を使って単一の値を生成します。
    • let numbers = [1, 2, 3, 4, 5];
let sum = numbers.reduce(function(total, num) {
    return total + num;
}, 0);
console.log(sum); // 15

10. DOM操作

JavaScriptを使用して、DOM（Document Object Model）を操作することができます。これにより、HTML要素を動的に追加、削除、変更することが可能です。

let element = document.getElementById("myElement");
element.textContent = "新しいテキスト";

11. イベント処理

JavaScriptを使用して、ユーザーの操作に応じてイベントを処理することができます。例えば、ボタンクリックやマウスオーバーなどのイベントを処理することができます。

let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("ボタンがクリックされました！");
});
