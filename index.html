JavaScriptの基本構文と特徴

JavaScriptとは？

JavaScriptは、Webページに動的な機能を追加するために使用されるプログラミング言語です。主にクライアントサイドで使用されますが、Node.jsを使うことでサーバーサイドでも利用できます。JavaScriptは、イベントの処理、ユーザーインターフェイスの操作、Web APIとの通信など、さまざまな用途に使われます。

基本的な構文

JavaScriptにはいくつかの基本的な構文があります。これらは他のプログラミング言語にも共通するものが多く、習得することでさまざまなプログラムを記述することができます。

1. 変数の宣言

変数はデータを格納するための入れ物です。JavaScriptでは、var、let、constを使って変数を宣言します。

// varを使った変数宣言
var message = "こんにちは、JavaScript!";

// letを使った変数宣言
let count = 10;

// constを使った定数の宣言
const pi = 3.14159;

2. データ型

JavaScriptのデータ型には、string（文字列）、number（数値）、boolean（真偽値）、object（オブジェクト）、undefined（未定義）、null（無）などがあります。

3. 条件分岐

条件分岐は、特定の条件が満たされた場合に異なるコードを実行するために使われます。if文とelse文がよく使われます。

let age = 20;

if (age >= 18) {
    console.log("成人です。");
} else {
    console.log("未成年です。");
}

4. 繰り返し処理

繰り返し処理は、同じコードを複数回実行するために使用されます。forループやwhileループが一般的です。

for (let i = 0; i < 5; i++) {
    console.log("ループ番号: " + i);
}

5. 関数

関数は、特定のタスクを実行するための再利用可能なブロックです。functionキーワードを使って関数を定義します。

function greet(name) {
    return "こんにちは、" + name + "!";
}

console.log(greet("太郎"));

6. オブジェクトと配列

JavaScriptでは、オブジェクトと配列を使って複数のデータをまとめて管理することができます。

// オブジェクト
let person = {
    name: "花子",
    age: 25,
    greet: function() {
        console.log("こんにちは、" + this.name + "です。");
    }
};

// 配列
let fruits = ["りんご", "バナナ", "オレンジ"];

console.log(person.name);
person.greet();

console.log(fruits[0]);
